# Python Practice Repository

This repository contains a collection of Python scripts and small projects organized by semester and study topics. It is intended for learning, practicing, and experimenting with Python programming concepts.

## Folder Structure

- **Semester_01/**: Introductory Python exercises and scripts covering basic programming concepts.
	- `76776ExamBookstore.py`, `BloodType.py`, `Boolean.py`, `Conditionals.py`, `Function.py`, `Inputs.py`, `Operations.py`, `TaxCalculator.py`, `While.py`, `WhileTask.py`, `conditionalClass.py`, `dict.py`, `lists.py`, `loops.py`, `maths.py`, `numberss.py`
- **Semester_02/**: Intermediate Python exercises, including object-oriented programming, data handling, and more advanced topics.
	- `76776_Gif.py`, `BikeRental_SS.py`, `CStudent.py`, `CStudentApp.py`, `CheckDate.py`, `Classes.py`, `GamesList.py`, `datadict.json`, `dictonary.py`, `hierachyClasses.py`, `import_jason.py`, `pet.py`, `plataform.py`, `stack.py`, `stream.py`, `vet.py`
- **Studies/**: Miscellaneous projects and experiments.
	- `GameStore.py`, `myqrcode.py`

## Usage

Browse the folders and open any Python file to explore different programming concepts and solutions. Each script is self-contained and can be run independently.
